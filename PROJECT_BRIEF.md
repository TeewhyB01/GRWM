# GRWM Project Brief

GRWM stands for Get Ready With Me.

GRWM is a global AI fashion styling platform.

The product includes:
1. A mobile app for users.
2. A web admin dashboard.
3. Firebase backend.
4. AI outfit recommendations.
5. Wardrobe upload.
6. Weather-based daily styling.
7. Occasion-based styling.
8. Realistic avatar and virtual try-on architecture.
9. Shopping recommendations.
10. Free and premium plans.
11. Affiliate revenue.
12. Multilanguage-ready architecture.

Core user problems:
1. Help users style clothes they already own based on weather, occasion, body shape, and personal taste.
2. Help users who do not know what to wear today.

Target users:
- Working women
- Young professionals
- Global users from day one
- English launch first, with multilingual architecture from day one

Important rules:
- Do not use npm.
- Use pnpm only.
- Do not rely on Expo Go.
- Use Expo development builds/EAS Build or a React Native setup that does not depend on Expo Go.
- Use TypeScript.
- Use Firebase Storage for user images.
- Use Firebase Authentication.
- Use Cloud Firestore.
- Use Firebase Cloud Functions.
- Keep the app privacy-first.
