# GRWM Product Requirements

## Product Summary

GRWM is a global, privacy-first fashion styling platform that helps users decide what to wear using their wardrobe, weather, occasion, body-shape preferences, and personal taste.

## Launch Audience

- Working women
- Young professionals
- Global users from day one
- English-first launch with multilingual architecture from day one

## MVP Scope

- Expo React Native mobile app built for EAS development builds.
- Account creation and sign-in through Firebase Authentication.
- Wardrobe item capture and metadata management.
- Daily styling context collection for weather and occasion.
- User profile preferences for style, fit, and privacy controls.
- Admin dashboard foundation for operational visibility.

## Explicitly Out Of Scope For This Foundation

- AI outfit recommendation logic.
- Realistic avatar generation.
- Virtual try-on.
- Payments and premium plan enforcement.
- Shopping and affiliate recommendation flows.
- Expo Go support.

## Success Criteria

- The repository can install, typecheck, lint, and test with pnpm.
- Product boundaries and future feature phases are documented.
- Shared TypeScript contracts exist for cross-app alignment.
- Firebase rule placeholders exist before data is collected.
